# xsshelp
写论文空余时间闲着没事写的一个辅助挖掘xss漏洞的工具（主要手懒为了省事，就把每回挖xss的一个思路简单给用工具实现了下）


```xml
xsshelp version: 1.0.0
Usage:  [-ut] [-u url] [-t thread] [-h help]

Options:
  -h	this help
  -t int
    	thread Num (default 8)
  -u string
    	a target url(Please add http or https)
```
